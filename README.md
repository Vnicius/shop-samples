# Shop Samples

## Iniciar Banco de Dados

Utilizar o script **scriptbd.sql** para a criação do Banco de Dados SQL.

## Iniciar Servidor

1. Instale as dependências
```
    $ npm install
```

2. Inicializar o servidor
```
    $ npm start
```

3. Abrir **http://localhost:3000**
